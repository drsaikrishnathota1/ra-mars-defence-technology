# Conclusion

This paper proposes RA-MARS, a resilient AI-driven mission assurance framework for secure multi-UAV defence surveillance in contested environments. The framework is designed to address three major adversarial conditions affecting UAV operations: radio-frequency jamming, GPS spoofing, and mission-data tampering.

Unlike approaches that focus only on isolated attack detection or secure logging, RA-MARS integrates AI-based attack detection, mission-risk scoring, adaptive mission-continuation logic, and blockchain-inspired tamper-resistant mission logging into a unified mission-assurance workflow. This integrated approach is intended to improve not only attack awareness but also operational continuity, communication reliability, and mission-data trustworthiness.

The proposed evaluation plan considers normal operation, RF jamming, GPS spoofing, data tampering, and combined attack scenarios. RA-MARS is compared against conventional UAV surveillance, AI-only detection, logging-only, and non-adaptive secure baselines. The evaluation metrics include mission success rate, attack detection accuracy, precision, recall, F1-score, packet delivery ratio, average latency, energy consumption, tamper-detection rate, and mission recovery time.

The expected contribution of RA-MARS is a defence-oriented framework that evaluates UAV resilience from a mission-assurance perspective rather than only from a cybersecurity or classification perspective. By linking detection outputs with risk-aware mission decisions and tamper-resistant records, RA-MARS aims to support secure and resilient UAV surveillance in contested defence environments.

Future work may include hardware-in-the-loop testing, real UAV flight experiments, larger-scale swarm scenarios, advanced adversarial AI attacks, integration with real communication protocols, and validation using realistic defence surveillance datasets.
