# Final Reference Verification Checklist

## Purpose
This checklist tracks final reference verification before submitting the RA-MARS manuscript.

## Required Checks

For each reference in `reference-list-draft.md`, verify:

- [ ] Author names are correct
- [ ] Article title is exact
- [ ] Journal/conference name is exact
- [ ] Year is correct
- [ ] Volume is correct
- [ ] Issue is correct, if applicable
- [ ] Page range or article number is correct
- [ ] DOI is correct
- [ ] Reference is not duplicated
- [ ] Reference is relevant to RA-MARS

## Final Reference Rules

- Use Elsevier numbered reference style.
- Do not include unverifiable references.
- Do not include weak references only to increase count.
- Avoid excessive MDPI references.
- Keep the final reference count around 43–50.
- Ensure every in-text citation number appears in the reference list.
- Ensure every reference list item is cited in the manuscript.

## Notes

The manuscript should not be submitted until all references and DOI values are verified from publisher pages, IEEE Xplore, ScienceDirect, ACM, Springer, or official DOI records.
